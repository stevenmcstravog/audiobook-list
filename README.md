# audiobooklist
A simple single page application that allows you to pull audiobook data from a mock API, add/edit/delete an audiobook and simple form validation.

### Tech
- React
- React Router
- Node.js
- gulp.js
- Browserify
- Bootstrap
- valid-url
- toastr
- ES5
